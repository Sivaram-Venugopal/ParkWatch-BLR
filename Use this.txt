You are continuing work on ParkWatch BLR, a submission for Tata
Technologies InnoVent (Problem Statement 1: parking-induced congestion
detection for Bengaluru). A prior agent (running in a cloud sandbox)
completed the full pipeline and dashboard. Your job is to run everything
locally, verify it, then close three specific gaps before final submission.

WHAT IS ALREADY DONE (do not redo, only verify it runs on your machine):

1. Data pipeline (pipeline.py): loads 298,445 raw violation records,
   removes 5,372 exact duplicates AND 49,799 confirmed-invalid records
   (validation_status = rejected/duplicate), parses multi-label violation
   arrays, computes a Congestion Impact Score (CIS) per violation as
   severity_weight x vehicle_weight x time_of_day_weight, with documented
   weights for all violation types and vehicle types.
2. Spatial hotspot zoning: fixed 150m x 150m grid cells (NOT density
   clustering — DBSCAN was tested and rejected after it chained 55% of
   all violations into one cluster spanning multiple police jurisdictions;
   this is documented in the README, don't re-introduce DBSCAN).
3. Enforcement Priority Index (EPI): 0.5 x volume + 0.3 x persistence +
   0.2 x severity ceiling, all min-max normalized. Produces 2,100
   qualifying hotspot zones, 630 Critical/High tier.
4. Predictive model: Random Forest regressor, R2 = 0.9063, MAE = 0.5933
   on a single 80/20 train-test split, predicting CIS from
   [lat, lon, hour sin/cos, day-of-week sin/cos, weekend flag, police
   station, vehicle type, zone id].
5. Dashboard (dashboard/index.html): fully self-contained, Leaflet +
   Chart.js vendored locally in dashboard/vendor/ (no CDN dependency for
   core libraries — map tiles still need internet). Opens directly via
   double-click, no server required. Map, searchable leaderboard, 4
   analytics charts, and a methodology/limitations panel are all working.
6. Submission package: README.md with run instructions, submission_text.md
   with Title/Description/Snapshot guidance/Video script, a 2,000-row
   sample CSV for the repo (full CSV is 105MB, near GitHub's limit).
